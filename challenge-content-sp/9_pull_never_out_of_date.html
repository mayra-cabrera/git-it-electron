Sincronizando desde un remoto

Si estás trabajando con alguien más, lo más probable es que necesites manterte al día con los últimos cambios. Eso significa que necesitas actualizar cualquier cambio que haya sido agregado al repositorio de Github

¿En qué ha estado Reborobot?

Revisa si Reporobot ha hecho algún cambio a tu rama bajando los cambios desde el remoto llamado 'origin' en Github

git pull <REMOTENAME> <BRANCHNAME>

Si no hay ningún cambio, aparecerá "Actualizado". Si hay cambios, Git mezclará esos cambios en tu versión local.

¿Reporobot hizo cambios? Git te dirá donde se hicieron los cambios. Puedes abrir ese archivo y verificar los cambios de Reporobot. Sorprendentemente, ¡Reporobot es un artista!
